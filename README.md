# BOSS 自动投递助手（无 AI）

一个用于 BOSS 直聘网页端的 Tampermonkey/油猴脚本，提供自动扫描岗位、自动投递、自动下滑加载、屏蔽词跳过、外企筛选、近期活跃 Boss 过滤和固定招呼语发送能力。

> 脚本不包含 AI 功能，不连接第三方服务器，也不设置自动更新地址。更新脚本需要手动复制安装。

## 文件

- `boss-auto-apply.user.js`：油猴用户脚本主文件。

## 安装

1. 在 Chrome/Edge 安装 Tampermonkey 或同类用户脚本管理器。
2. 打开 Tampermonkey 管理面板，创建新脚本。
3. 将 `boss-auto-apply.user.js` 的内容完整复制进去并保存。
4. 打开或刷新 BOSS 直聘页面：`https://www.zhipin.com/web/geek/*`。
5. 页面右上角会出现悬浮面板，可拖动、可收起。

## 功能

- **自动投递**：点击“开始投递”后，自动扫描当前岗位并调用 BOSS 页面接口发起沟通。
- **自动下滑加载**：适配 BOSS 当前无限下滑加载模式，不再依赖传统翻页按钮。
- **最大投递数**：达到设置的投递成功数量后自动停止。
- **固定招呼语**：投递成功后可发送固定招呼语。
- **屏蔽词跳过**：命中公司名、岗位名、地区或岗位原始信息中的屏蔽词时自动跳过。
- **只投外企**：开启后，只有命中外企关键词的岗位才会投递。
- **恢复默认外企关键词**：外企关键词误改后，可一键恢复默认列表。
- **过滤不活跃 Boss**：开启后，只投递近期活跃 Boss；活跃度未知会跳过。
- **跳过已沟通岗位**：遇到已打招呼/已沟通过的岗位会自动跳过，并继续向下查找。
- **本地保存设置**：配置保存在浏览器 `localStorage`，刷新后仍保留。

## 配置说明

### 最大投递数

控制本轮最多成功投递多少个岗位。脚本的主要停止条件就是这个数量。

### 间隔秒

每次投递后的等待时间。建议不要设置过低，避免触发平台频控。

### 屏蔽词

支持换行、逗号、空格、顿号等分隔，例如：

```text
外包
培训
销售
某某公司
```

命中后会跳过对应岗位。

### 只投外企

开启后，脚本会在岗位卡片、公司标签和接口返回信息中匹配“外企关键词”。未命中会跳过。

默认关键词包括：

如果发现某些外企没有被识别，可以把公司名、品牌名或关键词补充到外企关键词中。

### 过滤不活跃 Boss

开启后会尝试识别 Boss 活跃状态，只投递在线、今天、几分钟前、几小时前等近期活跃 Boss。无法识别活跃度时会跳过。

## 使用建议

- 先在 BOSS 页面手动设置城市、岗位关键词、薪资、经验等基础筛选条件。
- 不建议把投递间隔设得太短。
- 如出现验证、频繁、风控、登录异常等提示，脚本会尝试停止。
- 外企筛选依赖页面/接口可见文本，不保证 100% 准确，建议结合关键词自行维护。

## 注意事项

- 本脚本仅在当前登录账号和当前 BOSS 页面环境下运行。
- BOSS 页面结构或接口变化可能导致脚本失效，需要重新适配。
- 请合理使用，遵守平台规则。
